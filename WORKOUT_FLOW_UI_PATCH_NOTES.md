# Muscle Royalty — Workout Flow UI Patch

Workout-flow only.

## Changes

During workout:

- Top workout header `+ Add` button is now a simple `+` icon without a boxed button.
- Exercise cards only show the exercise name.
- Removed exercise category/status/source display.
- Removed target sets box.
- Removed target reps box.
- Removed exercise notes box.
- Removed Copy the last set.
- Add set button now stretches across the card and is centre aligned.
- After confirming workout end, the app returns to the Today page.

No Today dashboard changes.
No Cycle changes.
No Progress changes.
No Notion schema changes.
