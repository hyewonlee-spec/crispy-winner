# Muscle Royalty Phase 10.7 — Workout Flow Changes Only

This patch only changes workout flow.

## Changes

- Workout tab is hidden from the bottom menu unless an active workout is in progress.
- Starting a recommended workout asks: Are you ready to begin your workout?
- Build your own asks the same confirmation before starting.
- Confirming start creates the Workout Session entry in Notion immediately.
- During active workout, the app keeps the default palette but changes action accents to hot pink.
- Workout name is editable on the Workout tab.
- RPE removed from workout set UI and set saving.
- Duplicate last renamed to Copy the last set.
- Workout tab shows End Today’s workout while active.
- Ending workout asks: Are you sure?
- Yes records exercises/sets to the active Notion session.
- Completion message: Today’s workout has been recorded, WELL DONE!
- After ending, active workout mode stops and the Workout tab is hidden again.

No Today page layout changes. No Cycle page changes.
