# Muscle Royalty Phase 10.8 — Cycle Refinements Only

This patch only changes the Cycle page.

## Your cycle box

- Phase text is larger and centred.
- Period started button is reduced by about 50% and has more top padding.
- After confirming period start, the button changes to Period ended.
- Period ended opens the same date confirmation flow.

## Previous cycles

- Added a + button at top right.
- + opens a past cycle form with Start date and End date side by side.
- Save cycle adds the past cycle to the same Notion database: Cycle Logs.
- The row is saved with Period Start Date, Period End Date, Period Length, Cycle Length, Cycle Log title, and related cycle fields.
- The app shows up to 4 previous cycles.

## Cycle-aware check-in

- Flow options are arranged over two centred lines.
- Symptom buttons are smaller, two per line.
- Save today’s check-in is reduced by 30% and has more top padding.

No Today page changes. No Workout flow changes.
