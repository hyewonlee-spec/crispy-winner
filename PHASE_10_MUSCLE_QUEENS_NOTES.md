# Phase 10 — Muscle Queens Dashboard + Cycle/Workout UX

## App rename

The app is now named **Muscle Queens**.

## Today dashboard

Today is now a dashboard ordered as:

1. Are you ready for a workout?
2. Symptoms checklist
3. Colour-coded current condition
4. Cycle-aware training
5. Notes
6. Record the day
7. Workout

## Readiness check

- Renamed to: Are you ready for a workout?
- Date is shown as the current app date in Australian format.
- Date is display-only and cannot be edited.
- Dog walk minutes removed from Today.

## Cycle-aware training on Today

- Phase is larger and centred.
- Cycle day box removed.
- Next period box renamed: Next period is due.
- Training condition bubble is centred.
- Open Cycle tab uses a pastel light link-style box.

## Training condition bubble triggers

- Normal training: symptoms are low and cycle/readiness context is manageable.
- Technique / moderate strength: menstrual phase with active flow but symptoms are not high.
- Moderate intensity + recovery: luteal or symptom-heavy day with moderate sleep/mood/fatigue disruption.
- Reduce load 10–20%: symptom load is moderate/high, mood swings, bloating, indigestion, cramps, or fatigue are elevated.
- Recovery / rehab focus: heavy flow, high cramps, high fatigue, high sleep disruption, or headache/migraine is high.

## Cycle page

- Removed top Cycle-aware training dashboard from Cycle page.
- Period dates now shows current phase instead of “No active period recorded.”
- Period Started Today immediately marks today as period start.
- Cycle-aware check-in replaces Daily symptom check-in.
- Date label removed.
- Date field reduced in size.
- Flow is now bubble selection, up to two selections.
