# Muscle Royalty Install-Safe Base

This is a clean base package only.

## Purpose
A single deployable base for Vercel with pinned dependencies and a fresh package-lock.json.

## Included
- Stable v1 codebase
- Pinned dependency versions
- Fresh package-lock.json
- No extra feature patches layered on top

## Local verification
- npm install ✅
- npm run build ✅

## Recommended Vercel settings
- Framework Preset: Vite
- Install Command: npm ci
- Build Command: npm run build
- Output Directory: dist
- Node.js Version: 20.x
